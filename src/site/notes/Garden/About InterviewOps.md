# 🚀 About InterviewOps

**InterviewOps** — это сквозной (end-to-end) фреймворк и автоматизированный пайплайн для управления процессом поиска работы, ведения портфолио и тестирования AI-решений. 

Вместо хаотичного рассылания резюме, этот проект переводит рекрутинг на рельсы современных инженерных практик: управление знаниями как кодом (Knowledge as Code), автоматизация CI/CD и применение LLM/RAG для подготовки к интервью.
